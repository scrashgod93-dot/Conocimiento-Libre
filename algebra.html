Ecuaciones lineales

Una ecuación lineal tiene la forma general ax + b = c. El objetivo es aislar la variable x.

Ejemplo detallado: Resuelve 3(x - 2) + 5 = 2x + 11

Paso 1: Distribuye → 3x - 6 + 5 = 2x + 11 → 3x - 1 = 2x + 11
Paso 2: Resta 2x en ambos lados → x - 1 = 11
Paso 3: Suma 1 en ambos lados → x = 12

Verificación: 3(12 - 2) + 5 = 35 y 2(12) + 11 = 35 ✓

Polinomios y factorización

Un polinomio cuadrático tiene la forma ax² + bx + c = 0.

Fórmula de las raíces: x = [-b ± √(b² - 4ac)] / (2a)

Ejemplo: Resuelve x² + 5x + 6 = 0

a=1, b=5, c=6
Discriminante: b² - 4ac = 25 - 24 = 1
Raíces: x = [-5 ± √1]/2 → x = [-5 + 1]/2 = -2 y x = [-5 - 1]/2 = -3

Factorización: (x + 2)(x + 3) = 0 → raíces x = -2 o x = -3

Gráficamente, es una parábola que cruza el eje x en las raíces.

Sistemas de ecuaciones

Conjunto de ecuaciones con varias variables. Métodos comunes: eliminación y sustitución.

Ejemplo por eliminación:
3x + 2y = 7
2x - 2y = -2

Suma las ecuaciones → 5x = 5 → x = 1
Sustituye: 3(1) + 2y = 7 → 3 + 2y = 7 → 2y = 4 → y = 2

Solución: (x, y) = (1, 2)

Método gráfico: Las soluciones son los puntos donde se cruzan las rectas.
